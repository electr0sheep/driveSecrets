# driveSecrets

CLI tool to store secrets on Google Drive (cause why not?)

## Getting Started

### Dependencies

1. [Go](https://golang.org/)
2. [Cobra](https://github.com/spf13/cobra)
3. [Google API Go Client](https://github.com/googleapis/google-api-go-client)

### Run

```
go run main.go
```

### Build

```
go build main.go
```
